SPAGOBI TEST ENVIRONMENT

Questo ambiente ha lo scopo di supportare le fasi di TEST di SpagoBI e di fornire un primo repository di esempio di documenti analitici.

Esecuzione dei TEST

I test devono essere eseguiti seguendo la traccia dei fogli excel (https://web.eng.it/svn/spagobi/documentation/Test/piani) dove sono presenti riferimenti ai documenti da utilizzare per i TEST. I fogli excel rappresentano una check list, si consiglia di farne una copia e di utilizzarla per tracciare il risultato della fase di TEST.
